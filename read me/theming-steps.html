Drupal 8

Theming steps

  • 1.- Install Drupal 8 (Opcional Enviroment).
  • 2.- Theme setup.
    • 2.1- "mytheme".info.yml
    • 2.2- "mytheme".libraries.yml
  • 3.- Disable Cache and enable Twig Debug.
    • 3.1- Disable the cache; using the files inside the route sites/default
    • 3.2- Enable Twig debug addind the parameters: debug: true, auto-reload: true, cache: false, to the development.services.yml file.
